A++
===
Project for Client Side course in Haifa University, 2018

## Run
  0. Clone this repository and install dependencies  
     ```
     git clone https://github.com/ilben/clientside.git
     npm install
     ```
  1. Run the server  
     `node server/server.js <port>`
  2. Go to  
     `localhost:<port>`

Enjoy 😁
